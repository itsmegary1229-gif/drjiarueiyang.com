## 1. 技術規格

### 1.1 建站方式
- **純靜態網站**：HTML / CSS / JavaScript
- 不使用 CMS、不使用後端框架
- 不使用建站平台（不用 Wix、Squarespace 等）

### 1.2 託管與網域
- **託管平台**：GitHub Pages（免費）
- **網域**：`drjiarueiyang.com`
- **Repository 結構建議**：
  - `index.html`（首頁）
  - `about.html`（醫師介紹）
  - `blog/`（知識專欄）
  - `cases/`（案例分享）
  - `appointment.html`（門診掛號）
  - `assets/`（圖片、字型、CSS、JS）

### 1.3 技術選擇要求
- 程式碼**簡潔易讀**，方便日後業主請其他人協助維護
- **不使用** React、Vue、Next.js 等需要編譯的框架
- 可使用 vanilla JavaScript 或極簡前端工具
- CSS 建議使用單一 `style.css`（或拆成幾個邏輯檔案），不使用 Tailwind 編譯流程
- 字型透過 **Google Fonts CDN** 載入

### 1.4 響應式設計（RWD）
- **桌機版 + 手機版必須同步開發**
- 斷點建議：
  - 桌機：≥ 1024px
  - 平板：768px – 1023px
  - 手機：≤ 767px
- 手機版選單以漢堡選單（hamburger menu）呈現

### 1.5 語言
- **第一版：繁體中文**
- 預留英文版切換的架構（語言切換按鈕的 UI 預留位置，但功能可暫不實作）

## 3. 視覺設計系統

### 3.1 色彩規範

```css
:root {
  /* 主要色彩 */
  --color-bg: #F8F6F1;           /* 米白底色 */
  --color-primary: #26215C;      /* 深紫主色 */
  --color-accent: #B8A050;       /* 金色點綴 */

  /* 衍生色彩 */
  --color-primary-light: #3C3489;  /* 淺紫（hover 用） */
  --color-primary-pale: #EEEDFE;   /* 極淺紫（按鈕 hover 填色） */
  --color-text-primary: #1A1A1A;   /* 主要文字 */
  --color-text-secondary: #666666; /* 次要文字 */
  --color-text-tertiary: #999999;  /* 提示文字 */
  --color-border: rgba(184, 160, 80, 0.2);  /* 金色淡邊框 */
}
```

**色彩使用原則：**
- **米白 `#F8F6F1`** 為全站背景，營造優雅學術質感
- **深紫 `#26215C`** 用於：標題文字、主要按鈕外框、導覽列文字、重點內容
- **金色 `#B8A050`** 為點綴色，僅用於：分隔線、小裝飾、區塊小標、icon。**不大面積使用**

### 3.2 字型規範

**載入方式（Google Fonts）：**
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;500;600&family=Inter:wght@300;400;500&family=Noto+Serif+TC:wght@300;400;500&family=Noto+Sans+TC:wght@300;400;500&display=swap" rel="stylesheet">
```

**字型分配（核心原則：主角用 Serif，功能用 Sans）：**

| 用途 | 字型 |
|------|------|
| 英文標題 | Playfair Display |
| 中文標題 | Noto Serif TC |
| 英文內文 | Inter |
| 中文內文 | Noto Sans TC |

**Serif（Playfair Display + Noto Serif TC）使用場合：**
- 醫師姓名（楊佳叡醫師）
- 頁面 H1、H2 大標題
- 首頁定位語句
- 引言、名句
- 區塊小標（金色字）

**Sans（Inter + Noto Sans TC）使用場合：**
- 導覽列選項
- 內文段落
- 按鈕文字
- 日期、標籤、Hashtag
- 表格內容
- H3 子項目標題
- 所有功能性文字

### 3.3 字級層級

| 層級 | 用途 | 字型 | 桌機字級 | 手機字級 |
|------|------|------|---------|---------|
| H1 | 頁面主標題 | Playfair Display + Noto Serif TC | 44px | 32px |
| H2 | 區塊標題 | Playfair Display + Noto Serif TC | 30px | 24px |
| H3 | 子項目標題 | Noto Sans TC | 21px | 18px |
| 內文 | 正文段落 | Inter + Noto Sans TC | 16px | 15px |
| 小字 | 日期、標籤、備註 | Inter + Noto Sans TC | 13px | 12px |

**行高（line-height）：**
- 標題：1.3 – 1.4
- 內文：1.7 – 1.8（中文段落需要較寬行高）
- 小字：1.5

**字距（letter-spacing）：**
- 中文標題：0.02em
- 英文標題：normal
- 全大寫小標：0.1em – 0.15em

### 3.4 名字呈現方式

- **主標題**：「楊佳叡醫師」（Noto Serif TC，大字）
- **副標題**：「Jia-Ruei Yang, MD」（Playfair Display，小字，置於下方）

### 3.5 版面風格

- **整體風格**：中央對稱雜誌感（editorial / magazine layout）
- **最大內容寬度**：1100px，水平置中
- **左右留白**：桌機版 ≥ 24px，手機版 ≥ 16px
- **區塊垂直間距**：桌機 80–120px，手機 48–64px

### 3.6 按鈕規範

```css
/* 主要按鈕 */
.btn-primary {
  background: transparent;
  color: #26215C;
  border: 1px solid #26215C;
  padding: 12px 32px;
  font-family: 'Inter', 'Noto Sans TC', sans-serif;
  font-size: 14px;
  letter-spacing: 0.05em;
  transition: all 0.3s ease;
  border-radius: 0;  /* 方正不圓角，呼應雜誌感 */
}

.btn-primary:hover {
  background: #EEEDFE;
  border-color: #3C3489;
}
```

- **設計原則**：無填色外框線風格，hover 時填淡紫色
- **不要**使用陰影、漸層、圓角過大的按鈕

### 3.7 裝飾元素

- **首頁**：右側使用金色臉部線條插圖（呼應名片設計，SVG 格式）
- **內頁**：極簡金色點綴
  - 區塊分隔線：1px 金色細線
  - 小標前置裝飾：金色四角星 ✦ 或短線 ──
- **不使用**：陰影、漸層、模糊效果、霓虹光、複雜紋理

## 8. 效能與最佳實踐

### 8.1 圖片優化
- 所有照片需壓縮，桌機版主圖 ≤ 200KB，手機版 ≤ 100KB
- 使用 WebP 格式為主，加上 JPG fallback
- 加上 `loading="lazy"` 延遲載入

### 8.2 載入速度
- CSS 與 JS 檔案盡量精簡
- 字型載入使用 `font-display: swap`
- 不引入過多第三方套件

### 8.3 無障礙
- 所有圖片加 `alt` 屬性
- 表單欄位加 `label`
- 顏色對比符合 WCAG AA 標準（深紫 `#26215C` 在米白上對比足夠）
- 鍵盤可操作（Tab 鍵能到所有互動元素）

## 11. 設計風格參考總結

| 設計維度 | 採用方向 |
|---------|---------|
| 整體調性 | 優雅學術 × 專業權威 × 溫度兼具 |
| 色彩心理學 | 米白（純淨）× 深紫（信任、專業）× 金色（精緻、價值感） |
| 排版風格 | 雜誌編輯感、中央對稱、留白寬裕 |
| 字型氣質 | 標題用襯線（古典學術）、內文用無襯線（現代清晰） |
| 動效原則 | 克制簡約，僅使用淡入、滾動觸發、hover 微互動 |
| 不使用 | 漸層、陰影、霓虹、紋理、圓角過大、卡通插圖 |

業主是醫師，**沒有網頁設計經驗**，請特別注意：

1. **程式碼註解要清楚**，方便日後業主自己查看修改

5. 設計**克制優雅**勝過炫技，因為這是專業醫師形象網站
6. **不要使用任何「AI 感」設計**（紫色漸層、卡通 icon、空白堆疊式版面）
